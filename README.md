# Secure Task Manager

Full Stack MERN Task Management Application.

Features:
- User Register & Login
- JWT Authentication
- Create Tasks
- Edit Tasks
- Delete Tasks
- Mark Task Completed
- Dashboard Statistics

Tech Stack:
Frontend:
React
CSS

Backend:
Node.js
Express.js
MongoDB

How to run:

Backend
cd Back End
npm install
npm run dev

Frontend
cd Front End
npm install
npm start
